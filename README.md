# Mehakame — HTML Fundamentals

A structured collection of HTML practice files covering core web development concepts, from basic tags to SEO best practices.

## 📁 Repository Structure

```
├── entities/               # HTML character entities
├── form and input tags/    # Forms, inputs, labels, and related elements
├── id and class/           # Using id and class attributes for targeting elements
├── image/                  # Embedding and styling images
├── inline and block/       # Inline vs block-level elements
├── list/                   # Ordered, unordered, and description lists
├── seo and core web vitals/ # Meta tags, semantic HTML, and performance hints
├── video and audio/        # Embedding media with <video> and <audio>
├── index.html              # Main entry point
└── script.js               # JavaScript file
```

## 📚 Topics Covered

| Topic | Description |
|---|---|
| **Entities** | Special characters like `&amp;`, `&copy;`, `&lt;`, etc. |
| **Forms & Inputs** | `<form>`, `<input>`, `<textarea>`, `<select>`, `<button>` and more |
| **ID & Class** | Attribute selectors, styling hooks, and DOM targeting |
| **Images** | `<img>`, `src`, `alt`, `width`, `height`, and responsive images |
| **Inline & Block** | Differences between `<div>`, `<span>`, `<p>`, `<strong>`, etc. |
| **Lists** | `<ul>`, `<ol>`, `<li>`, `<dl>`, `<dt>`, `<dd>` |
| **SEO & Core Web Vitals** | `<meta>`, `<title>`, Open Graph tags, semantic elements |
| **Video & Audio** | `<video>`, `<audio>`, `controls`, `autoplay`, `<source>` |

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Mehakame/<repo-name>.git
   ```
2. Open `index.html` in your browser, or navigate into any topic folder and open its HTML file directly.

## 🛠️ Tech Stack

- **HTML5** — core markup
- **JavaScript** — minimal scripting via `script.js`

## 🎯 Purpose

This repo serves as a hands-on reference and practice ground for learning and revisiting HTML fundamentals. Each folder is self-contained and focused on a single concept, making it easy to study topics in isolation.
